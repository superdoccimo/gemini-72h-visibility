# Contributing（コントリビューション）

このリポジトリへの貢献を歓迎します。

## 歓迎する貢献

- **公式ソースの追加・更新** — Google のポリシー変更があれば Issue や PR で教えてください
- **翻訳** — 他の言語への翻訳
- **チェックリストの改善** — より実用的な対策の提案
- **誤りの指摘** — 事実誤認があれば遠慮なく Issue を立ててください

## お願い

- 「事実」と「意見」の線引きを維持してください（[`docs/claims.md`](docs/claims.md) 参照）
- 公式ソースへのリンクを添えてください
- 感情的な表現より、事実ベースの記述を優先してください

## Issue / PR

- Issue：疑問・提案・誤り報告
- PR：ドキュメントの修正・追加

---

Thank you for contributing!
