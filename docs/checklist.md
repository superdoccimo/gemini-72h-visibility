# Gemini プライバシー運用チェックリスト（個人向け）

## 目的

「設定を変えたつもりなのに、実際どうなるの？」を減らすためのセルフ監査。

---

## 1. 設定の確認（必須）

- [ ] Gemini Apps Activity（Keep Activity）の状態を確認した
- [ ] OFFにした場合、履歴の利便性が落ちることを理解した
- [ ] OFFでも最大72時間保存され得る、かつ Activity に出ない点を理解した（[公式に明記](https://support.google.com/gemini/answer/13278892?hl=en)）
- [ ] 自動削除の期間設定を確認した（ONの場合：3 / 18 / 36ヶ月から選択可能）

## 2. 入力ルール（事故防止）

- [ ] パスワード / APIキー / 秘密鍵 / 2FAコードを貼らない
- [ ] 顧客情報・個人情報（住所・電話・氏名・口座など）を貼らない
- [ ] 未公開の収益計画・設計・ログ（認証ヘッダ含む）を貼らない
- [ ] 必要ならマスク（伏せ字）してから貼る

## 3. 履歴が必要な場合の代替（おすすめ）

- [ ] 会話の要点をローカルに保存（Markdown / ノート / Obsidian 等）
- [ ] テンプレ化：質問文・プロンプトを使い回せる形で保管
- [ ] 「機密は別管理」を徹底（パスワードマネージャ等）

## 4. 業務利用の場合（別枠で検討）

- [ ] Workspace / Cloud のデータガバナンスを確認した
  - https://support.google.com/a/answer/15706919?hl=en
  - https://docs.cloud.google.com/gemini/docs/discover/data-governance
- [ ] 組織ポリシーに沿って利用している
- [ ] 管理者設定（会話履歴 / 保持期間）を最初に固めた
